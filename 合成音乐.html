    • 4.将生成的信号写入输出文件中,文件中需要写入的是wav格式,传递的参数有 : HZ频率 和 音阶

  • def synthesizer(freq, duration, amp=1.0, sampling_freq=44100):
    • '''
    • :param freq: 频率
    • :param duration: 音频时间长度
    • :param amp: 振幅
    • :param sampling_freq: 采样率
    • :return: 返回指定音调类型的数据
    • ''' # 创建时间轴 X

    • # 构建音频信号 ,参数 : 幅度和频率
    • audio = amp * np.sin(2 * np.pi * freq * t)
    • return audio.astype(np.int16)
